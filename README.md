# ADMINISTRADOR DE CONTRASEÑAS

## Qué es?

Pues como el nombre lo dice, es un administrador de contraseñas jaja.
Este especificamente sirve para generar contraseñas aleatorias y guardarlas. Tambien permite guardar contraseñas propias.

## Instalacion

1. Clonar el repositorio
```
git clone https://github.com/0plimplim0/PasswordManager
```
2. Crear el archivo data.json
3. Listo, ya puedes utilizar el programa.
